# MetaPlogging 전체 현황 대시보드

MetaPlogging 앱에서 수집되는 데이터를 통합 모니터링하기 위한 읽기 전용 대시보드입니다.

## 화면 구성

- 전체 가입자, 누적 이동 거리, 누적 활동 시간, 누적 수거량
- 시간대별·월별 데이터 수집량
- 사용자별 수거량 및 이동 거리
- 날짜별·사용자별 수거 위치 히트맵
- 좌표 누락, 사진 미첨부, 미완료 세션, GPS 노이즈 품질 지표

로그인 화면 없이 접속 즉시 전체 현황을 표시하며, 화면 디자인과 조작 방식은 기존 MetaPlogging 대시보드를 유지합니다.

## 현재 데이터 상태

현재 화면 데이터는 기존 대시보드의 샘플 데이터입니다. 실제 전체 데이터 연결에는 서버에 읽기 전용 대시보드 집계 API가 필요합니다. 현재 앱 API는 일부 전체 통계를 제외하면 로그인한 사용자 본인의 세션만 반환하므로, 프론트엔드에서 전 사용자 데이터를 직접 조회할 수 없습니다.

필요한 전체 집계 API 범위:

- 전체 요약 통계
- 기간별 수집량
- 사용자별 익명 집계
- 전체 수거 위치 좌표
- 데이터 품질 오류 건수 및 상세 목록

## 로컬 실행

```bash
python3 -m http.server 4173 --bind 127.0.0.1
```

그다음 `http://127.0.0.1:4173`을 엽니다.
